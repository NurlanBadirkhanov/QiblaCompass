# QiblaCompass

QiblaCompass is a free and open-source Android application built with **Jetpack Compose** that helps Muslims find the **Qibla direction** (towards the Kaaba in Mecca) using the phone's compass and GPS.

The app works offline after determining the user's location and provides a simple and clean interface.

## ✨ Features

- 📍 Detects user location using GPS
- 🧭 Compass sensor to detect device orientation
- 🕋 Shows Kaaba when the device is aligned with Qibla
- 🌍 Multi-language support (English, Russian, Azerbaijani, Arabic)
- ⚡ Smooth animations with Jetpack Compose
- 📱 Modern UI built entirely with Compose
- 🔓 Free and open source

## 🛠 Built With

- Kotlin
- Jetpack Compose
- Android Sensor API
- Google Play Location Services
